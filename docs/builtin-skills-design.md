# 基础内置技能设计说明

## 概述

以下三个技能是 Agent 的**基础能力层**，提供最底层的系统交互能力。它们通常由 DeepAgents 框架自带的工具覆盖，Skill 文件主要起补充说明和行为约束的作用。

---

## shell_command

**用途**：本地 Shell 命令执行

DeepAgents 自带 `execute` 工具可以执行 shell 命令，此 Skill 提供额外的使用规范和安全约束，例如：

- 长时间命令加 `timeout` 限制
- 避免交互式命令（如 `vim`、`top`），改用非交互模式（`top -bn1`）
- 管道和重定向的正确使用方式

---

## file_reader

**用途**：本地文件读取

教 Agent 如何读取和分析本地文件，包括：

- 大文件分块读取策略
- 二进制文件识别和跳过
- 编码检测（UTF-8 优先）

---

## code_explainer

**用途**：代码分析与解释

教 Agent 如何逐层解析代码结构：

- 从整体架构到具体函数
- 识别设计模式和关键逻辑
- 给出易于理解的中文解释
